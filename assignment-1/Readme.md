## Smart Contract Description

This smart contract is a simple **Input-Output contract** that allows the user to store and retrieve a number.

### 🔹 Contract Name:

Simple

### 🔹 Functions:

1. **setNumber(uint _num)**

   * Takes input from the user
   * Stores the value in the blockchain

2. **number()**

   * Returns the stored value
   * Acts as output function
